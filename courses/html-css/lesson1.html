الدرس 1: مقدمة في HTML

الوقت: 20 دقيقة المستوى: مبتدئ

ما هو HTML؟

HTML هي لغة ترميز تستخدم لإنشاء وتصميم صفحات ومواقع الويب. اختصار HTML يعني لغة ترميز النص الفائق (HyperText Markup Language).

HTML ليست لغة برمجة، بل هي لغة ترميز (Markup Language) تستخدم لوصف هيكل صفحة الويب.

أول صفحة HTML لك

لنبدأ بأبسط مثال لصفحة HTML:

مثال 1: هيكل HTML أساسي 
<!DOCTYPE html>
<html lang="ar" dir="rtl">
<head>
    <meta charset="UTF-8">
    <title>أول صفحة لي</title>
</head>
<body>
    <h1>مرحباً بالعالم!</h1>
    <p>هذه أول صفحة ويب أقوم بإنشائها.</p>
</body>
</html>

العناصر الأساسية في HTML

<h1> إلى <h6>

العناوين من الأهم (h1) إلى الأقل أهمية (h6)

<p>

لإنشاء فقرة نصية

<a>

لإنشاء رابط (رابط تشعبي)

<img>

لإدراج صورة

تمرين عملي

أنشئ صفحة HTML تحتوي على:

  • عنوان رئيسي باسمك
  • فقرة تقدم فيها نفسك
  • رابط لموقعك المفضل
<!DOCTYPE html>
<html lang="ar">
<head>
    <title>تعريفي</title>
</head>
<body>
    <h1>أنا [اسمك]</h1>
    <p>أنا أتعلم البرمجة وأحب تطوير مواقع الويب.</p>
    <a href="https://example.com">زيارة موقعي المفضل</a>
</body>
</html>
